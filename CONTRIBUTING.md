# Contribution Guidelines

Thank you for considering contributing to this project!

## Main Rules
- The project is licensed under **CC BY-NC 4.0**.
- All contributions must comply with this license: **commercial use is not allowed**.
- Keep the original attribution and indicate any changes.

## How to Contribute
1. Fork the repository.
2. Create a branch for your changes.
3. Submit a pull request clearly describing your modifications.

## Code of Conduct
- Respect other contributors.
- Keep the code clear and well-documented.
